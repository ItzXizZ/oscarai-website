# OscarAI Website

A modern, responsive pre-order website for OscarAI - the revolutionary AI-powered waste sorting system.

## 🚀 Features

- **Modern Design**: Clean, professional aesthetic with smooth animations
- **Responsive**: Mobile-first design that works on all devices
- **Interactive**: Engaging animations and interactive elements
- **Performance**: Optimized for fast loading and smooth user experience
- **Accessible**: Built with accessibility best practices

## 🛠️ Tech Stack

- **React 18** - Modern React with hooks
- **Vite** - Fast build tool and development server
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Smooth animations and transitions
- **React Hook Form** - Form handling and validation
- **Lucide React** - Beautiful, customizable icons

## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd oscarai-website
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000`

## 🎨 Design System

### Colors
- **Primary Blue**: `#2563EB` - Main brand color
- **Secondary Green**: `#10B981` - Success and sustainability
- **Accent Orange**: `#F59E0B` - Highlights and awards
- **Neutral Gray**: `#6B7280` - Text and subtle elements
- **Background Light**: `#F9FAFB` - Light backgrounds
- **Text Dark**: `#1F2937` - Main text color

### Typography
- **Font Family**: Inter (Google Fonts)
- **Headings**: Bold weights (700-900)
- **Body**: Regular weight (400)
- **Line Height**: 1.6 for readability

### Components
- **Buttons**: Three variants (primary, secondary, success)
- **Cards**: Consistent shadow and border radius
- **Sections**: Standardized padding and spacing
- **Animations**: Smooth transitions with Framer Motion
